设计原则

  • ✓ 协议标准化：统一通过 OPC UA 接入工控数据
  • ✓ 解耦采集与消费：NiFi 作为中间层缓冲
  • ✓ 可扩展：新增数据源只需加适配器
  • ✓ 高可用：NiFi 集群 + 数据缓存

关键技术选型

  • Apache NiFi — 可视化数据流编排
  • 自定义适配器 — 处理非标准协议
  • OPC UA — 工业标准通信协议
  • 时序数据库 — 高效存储传感器数据

Apache NiFi 核心设计

为何选择 NiFi？

  • ✓ 可视化拖拽编排数据流
  • ✓ 内置 300+ Processor（OPC UA、HTTP、DB...）
  • ✓ 数据流优先级 + 背压机制
  • ✓ 集群部署，水平扩展
  • ✓ 完整的数据血缘与审计

自定义适配器开发

  • NiFi Nar（NiFi Archive）插件机制
  • Java 实现，继承 Processor 基类
  • 处理非标准协议（海康 ISAPI、WinCC 私有接口）
  • 独立打包，热部署无需重启

纯软件接入方案 ✓ 无需额外硬件

条件项要求验证方式
WinCC 4.x OPC UA ✓ WinCC 4.x 内置 OPC UA Server 确认 OPC UA 端口（默认 4840）已开放
网络连通性 ✓ NiFi 服务器能访问 WinCC / Siemens 网络 ping + telnet 端口测试
OPC UA 安全配置 ✓ 证书认证 / 用户名密码 UaExplorer 连接测试
One-Systems API ✓ RTSP URL 可获取 VLC 播放 RTSP 流测试
海康 ISAPI ✓ 摄像头 HTTP API 可达 curl 调用 ISAPI 接口
时间同步 ✓ NTP 统一时钟 各服务器时间差 < 1s

✅ 纯软件方案优势

  • WinCC 4.x 已内置 OPC UA Server，无需额外硬件网关
  • NiFi 通过标准协议直接对接，采集软件部署在现有服务器或虚拟机上
  • 海康摄像头原生支持 RTSP / ISAPI，自定义适配器纯软件实现
  • 零硬件成本，部署周期短（1-2周）

如需硬件补充的条件 可选增强

🔴 场景一：网络隔离

WinCC 在工控网段，NiFi 在办公网段，无路由

  • 工业网关（如 Moxa、Siemens SCALANCE）
  • 双网卡：一面接 PLC/WinCC，一面接 NiFi
  • 协议转换 + 网络隔离
  • 预估成本：¥3,000-8,000 / 台

🔴 场景二：WinCC 版本不支持 OPC UA

旧版 WinCC 无 OPC UA，仅有 S7 通信

  • OPC UA 网关（如 Softing、Kepware）
  • 将 S7 / MPI / Profibus 转为 OPC UA
  • 预估成本：¥5,000-20,000 / 套

🟡 场景三：高性能采集需求

采集点位 > 5000，频率 < 100ms

  • 边缘计算节点（工控机）
  • 部署在工控网络内，本地预处理
  • 批量上报 NiFi，减少网络压力
  • 预估成本：¥8,000-15,000 / 台

🟡 场景四：视频存储扩容

72路摄像头，30天录像存储

  • NVR / 存储服务器
  • 72路 × 30天 ≈ 40-80TB（按 2Mbps 码率）
  • RAID 5/6 + 冗余电源
  • 预估成本：¥50,000-120,000

💡 推荐实施路径

  • Phase 1（1-2周）：纯软件方案 → 验证 OPC UA + RTSP 连通性
  • Phase 2（3-4周）：完善自定义适配器 → 全量数据接入
  • Phase 3（按需）：根据实际网络和安全需求补充硬件

部署架构建议

NiFi 集群部署

  • 3节点集群（高可用）
  • 每个节点：4C8G 起步
  • ZooKeeper 协调（3节点）
  • SSL 加密通信

自定义适配器部署

  • 打包为 .nar 文件
  • 放入 NiFi lib/ 目录
  • 热加载，无需重启
  • 版本化管理（Git）

数据存储选型

数据类型推荐

监控运维

  • NiFi 自带监控 UI

  • ELK 日志收集
  • 告警规则：采集延迟 / 断流 / 异常值

方案总结

✅ 纯软件方案（推荐优先）

  • WinCC 4.x 内置 OPC UA，无需硬件网关
  • NiFi + 自定义适配器覆盖全部数据源
  • 部署周期短，成本最低

⚠️ 硬件补充（按需评估）

  • 网络隔离场景 → 工业网关
  • 旧版系统 → OPC UA 网关
  • 高性能需求 → 边缘工控机
  • 视频存储 → NVR / 存储服务器

📊 采集规模

  • 设备：~137台/套（BMS+EMS+CCTV）
  • 数据点位：~1500+
  • 视频通道：72路
